今天早上看到朋友圈被小猪佩奇刷屏了，然后中午看到推送有用代码实现小猪佩奇的绘制过程，但是代码不全，所以自己尝试着实现了一下。
大致分两步：
1.手绘小猪佩奇，然后算出几个关键点的坐标点
2.鼻子、头、耳朵、眼睛、腮、嘴、身体、手、脚、尾巴

绘图使用turtle库，下面简单讲解几个重要函数：
setheading()   选择绘制方向（0-东、90-北、180-西、270-南）
goto（）     定位坐标
penup()    提起笔
pendown()    放下笔
fd()  向前绘画
color()  画笔颜色
fillcolor() 填充颜色
circle() 画圆

详细的功能及知识可以参考官方文档：https://docs.python.org/3/library/turtle.html。
